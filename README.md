# Biometric-Attendance-system-V2.0
The instructions that you need to follow: 
1.	The parts: 
i)	1* ESP32 
ii)	2* Fingerprint scanner 
iii)	3* OLED display 0.96in 
2.	Install the ESP boards on the Arduino IDE using the additional links: http://arduino.esp8266.com/stable/pac... https://dl.espressif.com/dl/package_e... 
3.	Include the these libraries: 
i)	https://github.com/jfturcot/SimpleTimer 
ii)	https://github.com/adafruit/Adafruit-... 
iii)	https://github.com/adafruit/Adafruit_... 
iv)	https://github.com/adafruit/Adafruit-... 
4.	Download the ZIP file: https://github.com/InfinityWorldHI/Bi... 
5.	Put your WiFi credentials(SSID, Password) and the device token with your computer IP or your Server domain in case you want to use a public one. 
6.	Install a local server like xampp server or wampserver: 
i)	Xampp: https://www.apachefriends.org/downloa... 
ii)	Wamp server: http://www.wampserver.com/en/#downloa... 
7.	Copy the biometric attendance folder into xampp/htdocs folder or into wamp/www folder. 
8.	Go to localhost/phpmyadmin in order to create the database. 
9.	Then go to http://localhost/biometricattendancev... to start.
